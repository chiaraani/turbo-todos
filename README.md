## Description
A demonstration of turbo rails with todos

## Run on your computer

Ruby version 3.1

```
git clone git@github.com:chiaraani/turbo-todos.git
cd turbo-todos
rails db:create
rails db:migrate
rails s
```

Visit on http://localhost:3000/
